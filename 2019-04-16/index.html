    1. Meta Chars

      • /, \, ^, $, +, -
      • +, *, ?
      • {max, min}: number of times (exact or range)
      • [], short version -> [a-z], [a-z0-9], [a-zA-Z0-9], use ^ to negate inside [] (even shorter: \d, \D, \w, \W, \s, \S)

    2. Method:

      • .scan
      • .sub replace the first matching
      • .gsub replace all matching
